## 📋 핵심 기능 설계

### 1. 개인 모드 (기본)

#### **습관 관리**
```
✅ 개인 습관 추가/체크
- 매일 물 2L 마시기
- 운동 30분
- 독서 20페이지

✅ 개인 통계
- 연속 달성일 (스트릭)
- 월간 달성률
- 달력 뷰

✅ 개인 알림
- 시간대별 리마인더
- 달성 축하 알림
```

### 2. 그룹 모드 (핵심 차별화)

#### **그룹 생성 & 관리**
```
✅ 그룹 만들기
- 그룹명 설정
- 그룹 아이콘 선택
- 초대 링크/코드 생성
- 최대 20명

✅ 그룹 종류
- 가족 👨‍👩‍👧‍👦
- 연인 💑
- 스터디 📚
- 운동 🏃‍♂️
- 프로젝트 💼
- 커스텀 🎯
```

#### **그룹 습관 공유**
```
✅ 공유 습관 생성
예시: "가족 - 저녁 식사 함께하기"
- 전체 공개 (모든 멤버가 봄)
- 상태 실시간 공유
- 누가 완료했는지 표시

✅ 개인 습관 공개 설정
- 비공개 (나만)
- 그룹 공개 (멤버들이 봄)
- 응원받기 (달성시 알림)
```

#### **그룹 할일 (Tasks)**
```
✅ 일회성 할일 등록
"병원 예약하기" → 아빠/엄마 누구나 가능
"우유 사기" → 먼저 보는 사람이
"택배 받기" → 집에 있는 사람이

✅ 할일 상태
- 대기중 ⏰
- 진행중 🔄 (누가 하는지 표시)
- 완료 ✅
- 만료 ❌

✅ 할일 속성
- 담당자 지정 (특정인/아무나)
- 마감일 설정
- 우선순위 (긴급/보통/나중)
- 반복 설정 (매주/매월)
```

### 3. 알림 & 커뮤니케이션

#### **스마트 알림**
```
✅ 개인 알림
- "💧 물 마실 시간이에요!"
- "🎉 7일 연속 달성!"

✅ 그룹 알림
- "엄마가 '병원 예약' 완료했어요"
- "아직 아무도 '쓰레기 버리기' 안 했어요"
- "오늘 가족 습관 달성률 80%!"

✅ 응원 알림
- "파트너가 운동 완료! 👏"
- "3명이 당신을 응원해요 💪"
```

#### **댓글 & 이모지**
```
✅ 습관/할일에 댓글
- "수고했어요!"
- "내가 대신 할게"

✅ 빠른 반응 (이모지)
- 👍 좋아요
- 💪 힘내요
- 🔥 대단해요
- ❤️ 사랑해요
```

### 4. 통계 & 분석

#### **개인 통계**
```
✅ 기본 통계
- 전체 달성률
- 습관별 달성률
- 최장 스트릭

✅ 그룹 내 순위
- 이번 주 1등 🥇
- 달성률 순위
- 스트릭 순위
```

#### **그룹 통계**
```
✅ 그룹 전체
- 오늘 달성률
- 주간/월간 트렌드
- MVP (가장 성실한 멤버)

✅ 그룹 습관별
- 참여율
- 평균 달성률
- 가장 인기있는 습관
```
